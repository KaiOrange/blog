发表于 2019-06-29 | 分类于 Canvas

globalAlpha

globalAlpha是设置全局的透明度，取值范围是0~1，0表示透明，1表示不透明。我们之前没有设置所绘制的图形的透明度，但是都是不透明的，所以猜都能猜出来默认值是1。它的值可以是字符串类型，也可以数值类型，我们直接在上面代码中第十行中加入代码context.globalAlpha=0.5;看到的效果如下：
